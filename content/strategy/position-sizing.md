## 为什么需要仓位管理

> 即使选股正确，错误的仓位管理也可能导致亏损。

## 基本原则

1. **分散投资** — 不把鸡蛋放在一个篮子里
2. **控制单只仓位** — 单只股票不超过总仓位的 20%
3. **保留现金** — 永远留有加仓的子弹
4. **动态调整** — 根据市场情况调整仓位

## 常见仓位管理方法

### 等额分配法

```
总资金 = 100万
持仓数量 = 5只
每只仓位 = 20万
```

### 凯利公式

```
f = (bp - q) / b
f = 最优仓位比例
b = 赔率
p = 胜率
q = 1 - p
```

### 金字塔加仓法

```
首次买入 → 30% 仓位
第一次加仓 → 20% 仓位
第二次加仓 → 20% 仓位
保留 → 30% 现金
```

## 仓位管理的纪律

1. 永远不满仓
2. 永远不加死码
3. 亏损时减少仓位
4. 盈利时可以适当加仓
5. 严格执行止损

## 实战建议

- 新手建议单只股票不超过 10%
- 可以用 ETF 作为底仓
- 保留 20-30% 现金应对极端行情
